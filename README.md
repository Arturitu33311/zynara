## ¿Qué es?
Zynara automatiza tareas con IA para usuarios finales. 
Infraestructura 100% self-hosted, pagos reales integrados 
y panel de administración propio.

No es un side project de tutorial. Es una startup.

## Stack
| Capa | Tecnología |
|------|-----------|
| IA | Gemini 2.5 Flash |
| Backend | Node.js · Docker · Docker Compose |
| Proxy | Nginx con SSL A+ |
| Seguridad | Cloudflare (HSTS · CSP · DMARC · DNSSEC) |
| Pagos | Stripe · NOWPayments (LTC) |
| Infra | Debian · Dell Latitude 5520 (self-hosted) |
